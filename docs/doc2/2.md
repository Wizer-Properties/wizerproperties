4. Workflow Automation
Go to Settings → Automation → Workflow Rules
4.1 New Lead Notification
Trigger: When Lead is Created
Condition: Lead Source = "Wizer Platform"
Actions:

Send email notification to assigned owner
Create task: "Follow up with new inquiry" (due in 1 hour)
4.2 Schedule Viewing Confirmation
Trigger: When Deal is Created
Condition: Stage = "Viewing Scheduled"
Actions:

Send email to Contact (confirmation template)
Send email to Deal Owner (viewing alert)
Create task: "Confirm viewing 24h before" (due 1 day before Visiting_Time)
4.3 Viewing Follow-Up
Trigger: When Deal Stage changes to "Viewing Completed"
Actions:

Send follow-up email to Contact (thank you + next steps)
Create task: "Get viewing feedback" (due in 24 hours)
Create task: "Send additional property options" (due in 3 days)
4.4 Stale Lead Alert
Trigger: Scheduled (daily)
Condition: Last Activity Time > 7 days AND Stage not in (Closed Won, Closed Lost)
Actions:

Send alert to Deal Owner
Create task: "Re-engage stale lead"
4.5 No-Show Handling
Trigger: When Viewing_Status changes to "No-show"
Actions:

Create task: "Contact no-show prospect"
Send re-engagement email (automated, optional)

7. Reports & Dashboards
Go to Reports or Analytics
7.1 Essential Reports to Create
1. Lead Source Performance

Metric: Leads by Source, Conversion Rate by Source
Purpose: See which channels bring quality leads

2. Pipeline Velocity

Metric: Average days in each stage
Purpose: Identify bottlenecks

3. Viewing Conversion

Metric: Viewings scheduled vs completed vs converted
Purpose: Track viewing effectiveness

4. Agent Performance

Metric: Leads assigned, viewings conducted, deals closed by user
Purpose: Team performance tracking

5. Property Interest Analysis

Metric: Most viewed properties, most inquired buildings
Purpose: Identify high-interest inventory
7.2 Dashboard Widgets
Create a dashboard with:

Open Deals by Stage (Funnel chart)
Leads Created This Week (Line chart)
Upcoming Viewings (List)
Stale Deals (>7 days no activity)
Top Properties by Interest (Bar chart)


8. Team & Permissions
Go to Settings → Users & Control
8.1 Role Setup
Administrator (Avi, Justina)

Full access to all modules
Can modify settings, workflows, templates
Access to all reports

Sales Rep (future hires)

Access to Leads, Contacts, Deals
Can only see own records + shared records
Cannot modify settings

Developer Account Manager (future)

Access to Accounts (developers), Deals
Limited Lead access
Can view reports for their territory
8.2 Territory Management (Optional, for later)
When you have multiple agents, set up territories:

By location (Bangkok, Phuket, Pattaya)
By property type (Condo, Villa, Commercial)
By developer (specific developer partnerships)

10. Maintenance Checklist
Weekly
Review new leads - are custom fields being populated?
Check stale deals - follow up or update status
Review upcoming viewings
Check workflow execution logs for errors
Monthly
Review lead source report - adjust marketing spend
Clean up duplicate records (Settings → Data Administration → Deduplicate)
Review and optimize email templates based on open rates
Check user activity - is everyone using the CRM?
Quarterly
Review pipeline stages - are they accurate?
Update lead scoring based on actual conversion data
Archive or merge old/dead leads
Review integration logs for any sync issues
